WhatsApp Web Translator

Translate messages you receive on WhatsApp Web using your own LLM. Nothing is sent anywhere except the provider you configure below.

Provider

Uses https://api.deepseek.com/v1 with deepseek-chat, which is fast and cheap and works well for chat translation.
claude-haiku-4-5 is the fastest and cheapest and works well for chat. For higher quality use claude-sonnet-5 or claude-opus-4-8.
Your API key is stored locally in the browser and is only sent to the provider above. A custom Base URL host will ask for a one-time permission when you save.

Style

Applies to both the messages you read and the drafts you translate. The model adapts wording to the context rather than translating word-for-word.

Reading (their messages)

You can toggle automatic mode on and off at any time from the small control inside the chat. The daily limit caps automatic translations per day so a busy group chat cannot run up your API bill; click-to-translate is never capped.

Writing (your drafts) - optional

Add a button to the message box that translates your draft before you send it.

Knowledge base - optional
